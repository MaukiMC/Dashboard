## Running (on mauki.net)
| **Key**         	| **Value**               	|
|-----------------	|-------------------------	|
| Location        	| `/home/www/minecraft`     |
| Screen Name     	| `mc-dash`               	|
| Open (detached) 	| `screen -r mc-dash`     	|
| Open (attached) 	| `screen -x mc-dash`     	|
| Execute from cl 	| `npm start`             	|
